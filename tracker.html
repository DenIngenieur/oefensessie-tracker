📖 User Guide – Practice Session Tracker

1. Create Question Series

Enter a name (e.g., "July 2025") and the total number of questions in the series. Click Add Question Series.

2. Select Active Series

Choose a series from the dropdown. Statistics and input forms will load automatically.

3. Add Question Results

Fill in the date (remembered), a unique Question ID (e.g., "Question 12"), the status (Good/Wrong/Blank/Not seen yet), and the time in minutes (optional). Click Add Result.

For status Wrong, Blank, or Not seen yet, a repeat date (3 days later) is automatically calculated.

4. View and Delete Results

The table shows all entered questions. You can delete a row using the Delete button.

5. Statistics

Track your progress: number of entered questions, percentages correct/wrong, average time, remaining, etc.

6. Export / Import

Use the buttons in the sidebar to back up all your data (JSON file) or restore a previous backup. Caution: Importing overwrites all existing data!

Good luck with your preparation! 🚀
